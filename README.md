📚 OVERVIEW
══════════════════════════════════════════════════════
HACKER'S LEGACY is an epic hacking RPG set in 2077.
You start as a nobody and rise to become a legend.
100 missions across 5 acts, with 3 paths and 9 endings.

🎮 HOW TO PLAY
══════════════════════════════════════════════════════
• Type 'help' to see all commands
• Type 'path' to choose your destiny (White/Black/Grey)
• Type 'missions' to see available missions
• Type 'start 1' to begin your journey
• Type 'status' to check your progress
• Type 'map' to see your journey map
• Type 'journal' to read your story log
• Type 'achievements' to track your medals
• Type 'slots' to save/load your game
• Type 'admin' for god mode

## 🏗️ GAME STRUCTURE — 5 ACTS, 100 MISSIONS

📜 THE STORY
══════════════════════════════════════════════════════
ACT I: THE AWAKENING (Missions 1-20)
  Chapter 1: The Message - A mysterious figure contacts you
  Chapter 2: The Underground - Learn the basics of hacking
  Chapter 3: The Truth - Discover who's really watching you
  Chapter 4: The Choice - Choose your path

ACT II: THE CONSPIRACY (Missions 21-40)
  Chapter 5: The Hunt - XSS attacks
  Chapter 6: CSRF - Cross-site request forgery
  Chapter 7: SSRF - Server-side request forgery
  Chapter 8: XXE - XML external entities

ACT III: THE WAR (Missions 41-60)
  Chapter 9: Command Injection - Execute OS commands
  Chapter 10: File Inclusion - LFI/RFI attacks
  Chapter 11: File Upload - Upload webshells
  Chapter 12: Wireless - WiFi hacking
  Chapter 13: Network - ARP, DNS, DHCP attacks

ACT IV: THE RECKONING (Missions 61-80)
  Chapter 14: Metasploit - Exploitation framework
  Chapter 15: Privilege Escalation - Get root/admin
  Chapter 16: Lateral Movement - Move through network
  Chapter 17: Persistence - Stay in the system

ACT V: THE ENDING (Missions 81-100)
  Chapter 18: Reverse Engineering - Analyze malware
  Chapter 19: Bug Bounty - Find vulnerabilities legally
  Chapter 20: The Final Choice - 9 unique endings
